# Jokenp-ou-Pedra-Papel-e-Tesoura-JOGO-
Oi! Este é o meu primeiro projeto de jogo interativo. Fiz ele durante o curso de Python para colocar em prática o que aprendi sobre condições e bibliotecas.

Lógica de Jogo: Usei if, elif e else para decidir quem ganha.
Sorteio: Usei a biblioteca random para o computador escolher a jogada.
Animação: Usei o time.sleep para dar um tempo entre o "JO... KEN... PO!".
Cores no Terminal: Criei um dicionário de cores para deixar o resultado mais fácil de ler (Verde para vitória e Vermelho para derrota).

Adicionei Arte ASCII para cada jogada, assim o terminal fica mais visual e divertido de jogar. Também tratei o erro de quando o usuário digita um número que não existe, para o programa não travar.
